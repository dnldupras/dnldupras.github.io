Inspirations & Aspirations

What Drives Me

The quote "work smarter not harder" has always been inspiring to me, if you can do something efficiently then you don't need to work as hard on it. A lot of computer science is working towards making efficient and clean work.

Through using the web for years I've always been inspired by clean and minimalistic designs that are oh so popular. I aspire to be able to make a clean but functional website.

The mars rovers have always been inspiring, seeing these robots and what they can do from so far away has always been amazing to me, I wish to be a be able to create something so awesome eventually.

Elon Musk is a huge inspiration for me, going from making paypal to being a tech giant with so many different projects that are for the betterment of humanity is a wonder to see.
